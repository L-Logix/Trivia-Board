How the Demo Works

1

Click a Clue

Select any clue value on the board. The trivia question appears in the display area below.

2

Read the Question

Think of your answer while the timer starts running — just like a real game show.

3

Answer Revealed

After a few seconds, the correct answer fades in. See if you got it right.

4

Player Buzzes In

A simulated player buzzes in automatically. Scores update live as players gain or lose points.

5

Continue or Reset

Click the next clue to keep playing. After all clues are used, a reset button lets you start fresh.

Data Collection Notice

Trivia Broadcast Engine collects anonymous usage statistics to help improve the software.

What is collected:

  • How many times each CLI command runs
  • How many games are started, completed, won
  • Aggregate counts of correct/incorrect answers
  • Feature usage counts (bonus clues, double rounds, etc.)
  • Setup mode and preset preferences

No personal data is collected. No category names, clues, answers, player names, IP addresses, or any identifiable information is ever recorded or transmitted.

These counters are stored locally and served from your trivia server. No data is sent to external services.
